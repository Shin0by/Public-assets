Privacy Policy

StarCitizen Hangar Helper Browser Extension

Last Updated: [Current Date]

1. Overview

StarCitizen Hangar Helper is a browser extension designed to enhance the user experience on the Roberts Space Industries (RSI) website. This privacy policy explains how we handle user data.

2. Data Collection and Storage

We do NOT collect, store, or transmit any personal user data to external servers. The extension operates entirely locally within your browser.

Key Point: All data processing happens locally on your device. No information is sent to our servers or any third-party servers.

3. Data Processing

The extension processes only the following types of data:

  • Publicly available ship and pledge information displayed on RSI website pages you visit
  • Transaction history from your RSI account pages
  • Page structure and content for functionality purposes

All data processing occurs locally in your browser. No data is sent to external servers.

4. User Account Information

The extension requires access to RSI website pages to:

  • Display your hangar and buyback information
  • Generate export files (CSV/JSON) locally
  • Execute "melt" operations through RSI's official API

Important Security Information:

When using the melt functionality, your RSI account password is requested temporarily to authenticate with RSI's API. The password is:

  • Never stored anywhere in the extension
  • Never logged to any file
  • Never transmitted anywhere except to RSI's official servers during the API request
  • Immediately discarded after the API request completes

5. Permissions Usage

The extension requires the following permissions:

  • Host permissions for robertsspaceindustries.com: To access and interact with RSI website pages
  • No other permissions are required

6. Third-Party Services

The extension uses the following libraries and services:

  • jQuery and DataTables libraries for UI functionality
  • Blob.js for file generation
  • All libraries are included locally within the extension

7. Data Export

All exported data (CSV/JSON files) are generated locally and saved only to your device. We have no access to these files.

8. Changes to This Policy

We may update this privacy policy. The "Last Updated" date will reflect changes. We recommend reviewing this policy periodically.
